## Abstract
Chain-of-Thought (CoT) prompting has shown promise in enhancing the reasoning capabilities of large language models (LLMs) on text-attributed graphs (TAGs). This work reframes CoT-based graph learning through the principle of clustering as reasoning, offering a $k$-means interpretation of how iterative reasoning operates over graph-structured data. We observe that existing graph CoT methods rely on disjoint architectures and fixed graph representations, limiting step-by-step semantic-topological interaction and interpretability. To overcome this limitation, we propose a unified framework named KCoT that integrates CoT reasoning with graph representation learning. Our key theoretical result reveals a formal mathematical correspondence between a Transformer block and the $k$-means algorithm, allowing reasoning to be interpreted as iterative assignment and update steps. Based on this insight, we introduce a Semantic Discriminating Prompt that explicitly formulates these steps as structured CoT reasoning, together with a structure-grounded alignment strategy to fuse topological priors with evolving thought-conditioned representations. Experiments on standard benchmarks demonstrate consistent improvements over state-of-the-art methods, validating clustering as a principled mechanism for CoT-based graph learning.

## 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：将 CoT 推理重新解释为 k-means 聚类过程。通过建立 Transformer 块与 k-means 算法之间的形式数学对应，使 LLM 的自我注意力层能够近似实现软 k-means 的分配-更新迭代，从而将自然语言推理从「黑箱」转变为可解释的几何优化过程。
- **关键技术细节**：
    - **① 理论等价性（Proposition 4.1）**：证明存在参数化使 Transformer 注意力权重构成软 k-means 分配的 ε-逼近；特定构造下 ε=0 精确成立。因此，堆叠自注意力层可等价于迭代聚类。
    - **② 语义区分提示 (Semantic Discriminating Prompt)**：
        - *分配步骤*：指令 LLM 作为语义过滤器，识别目标节点与候选邻居的共享方面，丢弃低相似度节点（模拟 k-means 的分配）。
        - *更新步骤*：指令 LLM 将选中邻居信息抽象总结为「简洁致密段落」，作为语义质心的更新（模拟 k-means 的质心更新）。
    - **③ 结构锚定的想法构建 (Structure-grounded Thought Construction)**：
        - 利用固定图拓扑（1/2 跳邻居）得到结构化邻居集合 N_str，同时利用当前表示空间的 KNN 得到推理诱导的邻居集合 N^(t)；分别对上两组邻居应用语义提示获得表征 T_str 和 T^(t)，再拼接为推理状态 z^(t)。
    - **④ 条件网络 (Condition-Net)**：
        - 对状态 z^(t) 生成推理矩阵 P^(t) ∈ R^{|V|×d}，以逐元素乘方式调节下一轮节点特征 X^(t+1) = P^(t) ⊙ X。

    - **⑤ 整体算法流程**：
        1. 初始化节点表示 H^(0) = GraphEncoder(X, G)；构建结构化想法 T_str。
        2. 对每轮推理 t = 1..M：
            - 重编码 H^(t-1) = GraphEncoder(X^(t-1), G)
            - KNN 检索 N^(t)
            - 构建想法 T_str, T^(t) 并编码
            - 生成状态 z^(t)，通过 CondNet 得 P^(t)
            - 更新 X^(t+1) = P^(t) ⊙ X
        3. 最终表示 H^(M) 用于下游任务（节点分类/链接预测）。
    - **⑥ 定理 4.4（语义-结构错位收缩）**：证明 CoT 迭代使错位度量 Δ_t 满足 Δ_{t+1} ≤ ρ Δ_t + ε，几何收敛到误差地板，说明 CoT 本质上是使 LLM 语义与 GNN 结构对齐的迭代机制。
    - **⑦ 推论 4.6**：指出经典 k-means 只改善语义紧凑性，不显式调整语义-结构错位；而 CoT 的错位收缩特性提供了比 k-means 更强的对齐机制。

## 6. 论文的主要结论与发现
1. **理论核心**：Transformer 自注意力可以精确实现（ε=0）k-means 的分配步骤，将 CoT 推理等同于迭代聚类过程。
2. **方法有效性**：KCoT 在几乎所有测试任务（三种协议 × 4 数据集 × 2 任务）上一致超过所有对比方法，包括 GraphGPT、LLaGA 等最先进 LLM 方法。例如在 ogbn-Arxiv 节点分类上达 79.25%（Single Focus），高出 LLaGA-HO 的 76.66% 约 2.6%。

4. **可解释性优势**：t-SNE 可视化显示节点表示随 CoT 步骤逐渐形成更清晰、更分离的聚类；inter/intra-class 距离比持续提升，与理论分析一致。
5. **结构-语义对齐**：定理 4.4 证明 CoT 迭代使语义-结构错位几何收缩，说明了 CoT 作为对齐机制而非简单「更聪明推理」的本质。
6. **鲁棒性**：KCoT 在不同 LLM 基座（Vicuna-7B, Llama2-7B, ChatGPT-4.1 nano）下均表现稳定且随更强大的 LLM 进一步提升。

## 7. 优点
