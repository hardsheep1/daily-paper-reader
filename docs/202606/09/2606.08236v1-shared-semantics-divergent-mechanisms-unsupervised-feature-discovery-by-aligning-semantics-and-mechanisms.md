## 摘要
随着大语言模型在高风险场景中越来越多地部署，对工具的需求日益增长，这些工具不仅要审计模型输出，还要审计产生输出的内部计算。电路分析是机制可解释性的核心方法，但它通常是目标条件的（target-conditioned），解释单个提示与所选补全的配对。这种目标条件的设置可能掩盖模型续写分布中的异质性。我们引入了分布级无监督特征发现，它同时使用语义内容和序列级机制归因对采样续写进行聚类，而无需手动指定目标输出。我们的方法使用语义嵌入和前缀到续写的归因签名来表示每个续写，然后优化一个权衡语义连贯性、机制一致性和聚类粒度的率失真目标。在聚类和引导分析中，发现的聚类暴露了单视图基线遗漏的续写模式，并提供了干预证据，表明聚类签名对应于可操作的机制因素。总的来说，我们的方法通过提供对模型续写分布背后机制的可扩展审计，补充了电路分析和行为评估。

## Abstract
As large language models are increasingly deployed in high-stakes settings, there is a growing need for tools that audit not only model outputs but also the internal computations that produce them. Circuit analysis is a central approach in mechanistic interpretability, but it is typically target-conditioned, explaining a single prompt paired with a chosen completion. This target-conditioned setup can obscure heterogeneity across a model's continuation distribution. We introduce distribution-level unsupervised feature discovery, which clusters sampled continuations using both semantic content and sequence-level mechanistic attributions, without manually specifying target outputs. Our method represents each continuation with a semantic embedding and a prefix-to-continuation attribution signature, then optimizes a rate-distortion objective that trades off semantic coherence, mechanistic consistency, and cluster granularity. Across clustering and steering analyses, the discovered clusters expose continuation modes that single-view baselines miss and provide interventional evidence that cluster signatures correspond to actionable mechanistic factors. Overall, our approach complements circuit analysis and behavioral evaluation by providing a scalable audit of the mechanisms underlying a model's continuation distribution.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：传统电路分析（circuit analysis）是机制可解释性（mechanistic interpretability）的中心方法，但它通常**目标条件化（target-conditioned）**，即只针对一个特定的提示-补全对（prompt-completion pair）进行解释。这种单目标分析存在两大局限：
  - **泛化性不足**：对于同一个提示，不同语义相似的续写（continuations）可能依赖不同的内部机制，单目标分析无法揭示这种异质性。
  - **掩盖异质性**：即使输出语义相似，其内部的激活模式也可能截然不同，单目标分析会掩盖这种多样性。
- **整体含义**：为了更全面地审计大语言模型（LLM）的可靠性，需要一种**分布级（distribution-level）**的无监督分析方法，能够自动发现模型在续写分布中存在的多种潜在模式（latent modes），同时兼顾语义内容和机制归因。

## 2. 论文提出的方法论

- **核心思想**：利用**率失真理论（rate–distortion theory）**，在聚类过程中同时优化语义连贯性和机制一致性，从模型的续写分布中无监督地发现可解释的特征簇。
- **三阶段流程**：
  1. **续写采样（Continuation Sampling）**：对于给定前缀，使用核采样（nucleus sampling）生成 N 个不同的续写，并为每个续写分配一个长度归一化的概率权重 \(P_n\)。
  2. **双重表示（Dual Representation）**：每个续写由两个互补视图表示：
     - **语义嵌入（Semantic embedding，\(e_n\)）**：使用预训练嵌入模型提取，L2归一化。
     - **机制嵌入（Mechanistic embedding，\(a_n\)）**：基于转码器（transcoder）计算前缀中的稀疏特征对续写每个token的线性归因，然后对续写跨度取平均得到序列级归因向量，RMS归一化。
  3. **联合聚类（Joint Clustering via Rate–Distortion）**：定义一个率失真目标函数 \(L_{\text{RD}} = H(C) + \beta_e D^{(e)} + \beta_a D^{(a)}\)，其中：
     - \(H(C)\) 是聚类熵（率项），鼓励使用少量簇。
     - \(D^{(e)}\) 是语义失真（L2距离），\(D^{(a)}\) 是机制失真（L1距离）。
     - \(\beta_e = \gamma\beta\), \(\beta_a = (1-\gamma)\beta\)，\(\beta\) 控制整体率失真权衡，\(\gamma\) 平衡两个视图。
   - 优化采用**交替最小化（alternating minimization）**：E-step按目标函数更新分配，M-step更新簇中心（语义用加权均值，归因用加权中位数）。
   - **自适应操作**：包含**分裂（Split）**和**合并（Junk）**操作，根据信息论准则自动调整簇的数量，避免预先指定K。

- **方法创新**：首次将率失真框架引入机制可解释性，同时考虑语义和机制两个视图进行无监督续写模式发现，避免了手工指定目标输出。
- **自适应聚类**：通过分裂/合并操作自动决定簇数，而非固定K，更符合实际分布结构。

## 8. 不足与局限
